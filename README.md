# Phone-to-3D-Project
Computer Vision Project

Setup Procedure:
1) Create environment from root: In VS Code Terminal run, python -m venv venv
2) Activate:run, venv\Scripts\activate(Windows), source venv/bin/activate(Mac,Linux)
3) Install Requirements: Run pip install open3d numpy opencv-python after that pip freeze > requirements.txt
4) Install COLMAP from https://github.com/colmap/colmap/releases
